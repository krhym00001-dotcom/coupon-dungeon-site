붕괴: 스타레일 생방송 코드 즉시 입력 가이드 — 성간여정 최대한 모으는 법

📅 2026.04.09 · 🎮 붕괴: 스타레일

붕괴: 스타레일의 생방송 쿠폰 코드는 매 버전마다 개최되는 공식 생방송에서 3개씩 공개돼요. 각 코드당 보통 성간여정 1~2개와 크레딧, 연료 등이 포함되어 있어 놓치면 아까운 무료 보상이에요.

생방송 코드의 유효 기간은 대부분 48시간 이내예요. 생방송을 실시간으로 보지 못했더라도 커뮤니티나 쿠폰던전에서 코드를 빠르게 확인하면 충분히 기회 안에 입력할 수 있어요.

코드 입력 방법은 두 가지예요. 첫 번째는 HoYoLAB 공식 사이트나 앱에서 직접 입력하는 방법이고, 두 번째는 게임 내 설정 메뉴의 교환 코드 탭에서 입력하는 방법이에요. 두 방법 모두 동일한 결과를 제공하며, 보상은 게임 내 우편함으로 전달됩니다.

코드 입력 시 주의사항이 있어요. 코드는 계정당 1회만 사용 가능하고, 이미 입력한 코드를 다시 입력하면 오류가 발생해요. 또한 서버가 다르면 같은 코드라도 유효하지 않을 수 있으니 자신의 서버 코드인지 확인하세요.

쿠폰던전에서는 생방송이 진행되는 즉시 코드를 등록하고 있어요. 특히 붕괴: 스타레일 페이지를 북마크해두면 코드를 가장 빠르게 확인할 수 있습니다!
