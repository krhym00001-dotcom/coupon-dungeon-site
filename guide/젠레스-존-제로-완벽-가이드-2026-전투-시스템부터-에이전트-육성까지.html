젠레스 존 제로 완벽 가이드 2026 — 전투 시스템부터 에이전트 육성까지

📅 2026. 4. 27. · 🎮 젠레스 존 제로

젠레스 존 제로(ZZZ)는 호요버스가 개발한 액션 RPG로, 2024년 출시 이후 빠르게 인기를 얻고 있는 게임이에요. 현대적인 도시 배경과 화려한 액션 전투가 특징이며, 원신, 붕괴: 스타레일과 함께 호요버스의 주요 타이틀이 됐습니다.

게임 기본 시스템

속성 시스템
젠레스 존 제로는 6가지 속성이 있어요.
- 전기: 감전 상태 이상 유발
- 불: 연소 상태 이상 유발
- 얼음: 빙결 상태 이상 유발
- 물리: 충격 상태 이상 유발
- 에테르: 분열 상태 이상 유발
- 얼음불기: 특수 상태 이상

각 속성의 상태 이상을 활용하는 것이 전투의 핵심입니다.

충격 시스템
적의 충격 게이지를 0으로 만들면 격파 상태가 되어 추가 대미지를 입힐 수 있어요.

특수 스킬 시스템
각 에이전트의 특수 스킬을 올바른 타이밍에 사용하는 것이 중요합니다.

에이전트(캐릭터) 가이드

에이전트 역할 분류
- 공격형: 높은 피해량, 메인 딜러
- 방어형: 높은 방어력, 탱커 및 보호막 생성
- 지원형: 팀 버프, 힐러

S등급 에이전트 추천

전기 속성:
- 나이트 오울: 전기 딜러, 광역 전격 스킬
- 조에: 전기 서포터, 팀 버프 전문

불 속성:
- 루시: 불 딜러, 강력한 폭발기
- 그레이스: 불 서포터, 연소 스택 생성

물리 속성:
- 빌리: 물리 딜러, 빠른 연타 스타일
- 안톤: 물리 탱커, 아군 보호

A등급 활용 팁
A등급 에이전트도 잘 육성하면 충분히 S등급을 보조할 수 있어요. 4가지 스킬을 최대한 활용하는 것이 중요합니다.

전투 시스템 심층 분석

기본 콤보 시스템
젠레스 존 제로의 전투는 콤보 연결이 핵심이에요.

기본 공격 콤보 방법:
1. 일반 공격으로 충격 게이지 감소
2. 특수 스킬 타이밍에 발동
3. 지원 스킬로 추가 피해
4. 극한 회피로 적의 공격 무력화

극한 회피 시스템
적의 공격 직전에 회피하면 극한 회피가 발동돼요. 슬로우 효과와 함께 반격 기회를 얻습니다.

체인 공격 시스템
팀원들 간의 체인 공격 연결이 핵심이에요. 스킬 사용 후 다음 에이전트로 교체하면 체인 공격이 발동됩니다.

뱅붕 시스템(인터 녹)

젠레스 존 제로에는 비디오 대여점(뱅붕)을 운영하는 독특한 콘텐츠가 있어요.

뱅붕 운영 팁
- 고객의 취향에 맞는 VHS 제공
- 직원 육성으로 서비스 효율 증가
- 특별 손님 응대로 특별 아이템 획득

뱅붕의 중요성
뱅붕 운영을 통해 각종 업그레이드 재료와 폴리크롬을 획득할 수 있어요.

폴리크롬 (뽑기) 효율화

폴리크롬 획득 방법
- 스토리 진행 보상
- 일일 미션 완료
- 이벤트 참여
- 뱅붕 운영 보상
- 쿠폰 코드 입력

폴리크롬 소비 전략
- 항상 한정 배너 시 소비
- 천장 시스템 활용 (90회 이내 S등급 확정)
- 신규 에이전트보다 필요한 에이전트 위주

음파 디스크(성유물) 시스템

음파 디스크 세트 추천

공격형 에이전트:
- 카르마 공명 세트: 치명타 확률 + 피해 증가
- 차원 파열 세트: 충격 전문 딜러

지원형 에이전트:
- 정온 회복 세트: 힐 + 아군 강화
- 조화 공명 세트: 버프 지속시간 증가

쿠폰 코드 입력 방법

젠레스 존 제로는 공식 생방송과 이벤트에서 쿠폰 코드를 배포해요.

입력 방법: 게임 내 설정 → 계정 → 교환 코드 → 코드 입력 → 우편함 확인

생방송 코드는 24~48시간 내 만료되므로 즉시 입력하세요! 쿠폰던전에서 최신 젠레스 존 제로 쿠폰 코드를 항상 확인하세요!
